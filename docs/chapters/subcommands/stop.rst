stop
====

Stop jail(s).

.. code-block:: shell

  ishmael ~ # bastille stop folsom
  [folsom]:
  folsom: removed

.. code-block:: shell

  ishmael ~ # bastille stop help
  Usage: bastille stop [option(s)] TARGET
    Options:

    -v | --verbose              Print every action on jail stop.
    -x | --debug                Enable debug mode.
