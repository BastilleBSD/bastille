rename
======

Rename a jail.

.. code-block:: shell

  ishmael ~ # bastille rename azkaban arkham

.. code-block:: shell

  ishmael ~ # bastille rename help
  Usage: bastille rename [option(s)] TARGET NEW_NAME

      Options:

      -a | --auto           Auto mode. Start/stop jail(s) if required.
      -x | --debug          Enable debug mode.
